Text manipulation scenarios aligned with the "Working with Text" docs. All
scripts operate on `examples/Showcase.pdf` and emit results to
`output/working-with-text/`.

- `01_find_and_replace.ts` – replace the first paragraph with a new message.
- `02_redact_phrases.ts` – delete paragraphs containing forbidden phrases.
- `03_highlight_matches.ts` – recolor matches for a regex pattern.
- `04_move_text.ts` – reposition a paragraph using absolute coordinates.
- `05_change_font.ts` – restyle the document title.
- `06_add_watermark.ts` – apply a DRAFT watermark across pages.
